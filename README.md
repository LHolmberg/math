    - Rotations
    - Scale
    - Translation
    - Perspective
    - LookAt
    - Inverse
    - Conversions
    - Operations
  - 4D, 3D & 2D vectors
    - Normalize
    - Crossproduct
    - Dotproduct
    - Conversions
    - Operations
